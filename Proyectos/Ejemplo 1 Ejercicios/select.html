Select

El Select nos ayuda a seleccionar mediante un solo clik, una de varias opciones. Es supremamente util en cuestionarios de opciones ya predispuestas, como por ejemplo ciudades, instituicones o categorias.

El siguiente código muestra la estructura básica de un Select:

1 <input List="">
2       <datalist id="">
3             <option value="opción 1">
3             <option value="opción 2">
4             <option value="opción 3">
5       </datalist>

Hay muchas formas de crear Select, pero segun el curso de platzi, esta es una forma muy acertada de hacerla. Aquí justo adelante voy a colocar un ejemplo que brindo platzi:

1 <input List="cursos">
2       <datalist id="cursos">
3             <option value="JavaScript">
3             <option value="HTML5">
4             <option value="CSS3">
5       </datalist>
